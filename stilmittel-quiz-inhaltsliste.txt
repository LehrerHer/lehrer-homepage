STILMITTEL-QUIZ – Vollständige Inhaltsliste
==============================================================
  39 Stilmittel in 5 Kategorien
  ★ Mittelstufe = Klasse 5–10 | ○ Oberstufe = ab Klasse 10

╔══ BILDLICHKEIT (8 Stilmittel)

  ┌─ Metapher  [★ Mittelstufe]
  │  Erklärung : Bildlicher Ausdruck ohne Vergleichswort – ein Wort wird aus seinem eigentlichen Bedeutungsbereich in einen anderen übertragen.
  │  Beispiel  : „Jemandem das Herz brechen“  ‖  „Das Feuer der Leidenschaft“
  │  Wirkung   : veranschaulicht, verdichtet
  └──────────────────────────────────────────────────────────

  ┌─ Vergleich  [★ Mittelstufe]
  │  Erklärung : Verknüpfung zweier Bedeutungsbereiche durch „wie“ oder „als“.
  │  Beispiel  : „Er kämpft wie ein Löwe“  ‖  „Stärker als der Tod“
  │  Wirkung   : veranschaulicht, verbindet
  └──────────────────────────────────────────────────────────

  ┌─ Personifikation  [★ Mittelstufe]
  │  Erklärung : Menschliche Eigenschaften werden auf Nichtmenschliches übertragen.
  │  Beispiel  : „Die Sonne lacht“  ‖  „Der Wind flüstert“
  │  Wirkung   : verlebendigt, vermenschlicht
  └──────────────────────────────────────────────────────────

  ┌─ Symbol  [★ Mittelstufe]
  │  Erklärung : Ein konkreter Gegenstand steht für eine abstrakte Idee, oft kulturell festgelegt.
  │  Beispiel  : Rose = Liebe  ‖  Taube = Frieden
  │  Wirkung   : verdichtend, kulturell
  └──────────────────────────────────────────────────────────

  ┌─ Allegorie  [○ Oberstufe ]
  │  Erklärung : Fortgesetzte Metapher – ein abstraktes Konzept wird als konkretes Bild dargestellt.
  │  Beispiel  : „Sensenmann“ als Bild für den Tod  ‖  „Justitia“ mit Waage und Augenbinde
  │  Wirkung   : versinnbildlicht Abstraktes
  └──────────────────────────────────────────────────────────

  ┌─ Metonymie  [○ Oberstufe ]
  │  Erklärung : Ersetzung eines Begriffs durch einen sachlich verwandten (räumlich, zeitlich, ursächlich).
  │  Beispiel  : „Brecht lesen“ (statt: ein Werk von Brecht)  ‖  „Das Weiße Haus erklärte …“
  │  Wirkung   : verdichtet, ersetzt Begriff durch verwandten
  └──────────────────────────────────────────────────────────

  ┌─ Synekdoche  [○ Oberstufe ]
  │  Erklärung : Ein Teil steht für das Ganze (pars pro toto) oder das Ganze für einen Teil (totum pro parte) — Ersetzung innerhalb desselben Begriffsfelds
  │  Beispiel  : „Pro Nase zwei Würste“ (Nase = Person)  ‖  „Deutschland hat gewonnen“ ( = die Mannschaft)  ‖  „Solange du deine Füße unter meinen Tisch stellst …“ (Tisch = Haushalt)
  │  Wirkung   : verknappend, verallgemeinernd, eindringlich
  └──────────────────────────────────────────────────────────

  ┌─ Chiffre  [★ Mittelstufe]
  │  Erklärung : Verschlüsseltes Zeichen, das ohne zusätzliches Wissen nicht eindeutig zu entschlüsseln ist.
  │  Beispiel  : „JHWH“ in der hebräischen Bibel  ‖  „Schwarzschimmel nacht“ (Celan)
  │  Wirkung   : verrätselt, lädt mit Bedeutung auf
  └──────────────────────────────────────────────────────────

╔══ WORTFIGUR (6 Stilmittel)

  ┌─ Hyperbel  [★ Mittelstufe]
  │  Erklärung : Starke Übertreibung eines Sachverhalts.
  │  Beispiel  : „todernst“  ‖  „fuchsteufelswild“  ‖  „ein Meer von Tränen“
  │  Wirkung   : übertreibt, intensiviert
  └──────────────────────────────────────────────────────────

  ┌─ Litotes  [○ Oberstufe ]
  │  Erklärung : Hervorhebung eines Begriffs durch doppelte Verneinung oder Verneinung des Gegenteils
  │  Beispiel  : „nicht selten“  ‖  „nicht unüblich“  ‖  „Er hat damit nicht Unrecht.“
  │  Wirkung   : bekräftigend durch Umweg, erzeugt einen leicht ironischen Ton
  └──────────────────────────────────────────────────────────

  ┌─ Periphrase  [○ Oberstufe ]
  │  Erklärung : Umschreibung eines Begriffs durch kennzeichnende Eigenschaften.
  │  Beispiel  : „Halbgötter in Weiß“ (statt: Ärzte)  ‖  „der Allmächtige“ (statt: Gott)
  │  Wirkung   : umschreibend, charakterisierend
  └──────────────────────────────────────────────────────────

  ┌─ Onomatopoesie  [○ Oberstufe ]
  │  Erklärung : Sprachliche Nachahmung von Geräuschen oder Klängen.
  │  Beispiel  : „Knall“  ‖  „Miau“  ‖  „rumpeln“  ‖  „zischen“
  │  Wirkung   : lautmalerisch, klangnachahmend
  └──────────────────────────────────────────────────────────

  ┌─ Emphase  [○ Oberstufe ]
  │  Erklärung : Nachdrückliche Hervorhebung eines Wortes oder Ausdrucks.
  │  Beispiel  : „Menschen! Menschen!“ (Schiller)  ‖  „DAS ist Kunst!“
  │  Wirkung   : betont, emotional aufgeladen
  └──────────────────────────────────────────────────────────

  ┌─ Euphemismus  [★ Mittelstufe]
  │  Erklärung : Beschönigende Umschreibung eines unangenehmen oder anstößigen Sachverhalts.
  │  Beispiel  : „entschlafen“ (statt: sterben)  ‖  „freisetzen“ (statt: entlassen)  ‖  „kollateraler Schaden“ (statt: getötete Zivilisten)
  │  Wirkung   : abmildernd, verhüllend — kann schützen oder Mitgefühl erzeugen, aber auch gezielt verschleiern und manipulieren (besonders in Politik und Medien)
  └──────────────────────────────────────────────────────────

╔══ GEDANKENFIGUR (8 Stilmittel)

  ┌─ Understatement  [○ Oberstufe ]
  │  Erklärung : Bewusste Untertreibung eines Sachverhalts — etwas wird als kleiner, harmloser oder weniger bedeutsam dargestellt, als es tatsächlich ist
  │  Beispiel  : „Das war nicht ganz optimal.“ (nach einer Katastrophe)  ‖  „Ich bin nicht ganz unglücklich darüber.“ (bei großer Freude)  ‖  „Das könnte schwierig werden.“ (wenn etwas fast aussichtslos ist)
  │  Wirkung   : abschwächend, oft ironisch oder humorvoll — kann Distanz, Coolness oder Bescheidenheit ausdrücken
  └──────────────────────────────────────────────────────────

  ┌─ Antithese  [★ Mittelstufe]
  │  Erklärung : Gegenüberstellung gegensätzlicher Begriffe oder Gedanken.
  │  Beispiel  : „Klein ist der Strauch, groß ist der Baum“  ‖  „Krieg und Frieden“ (Tolstoi)
  │  Wirkung   : kontrastierend, gegenüberstellend
  └──────────────────────────────────────────────────────────

  ┌─ Ironie  [★ Mittelstufe]
  │  Erklärung : Das Gegenteil des Gemeinten wird gesagt — die eigentliche Bedeutung ergibt sich erst aus dem Kontext, dem Tonfall oder dem Vorwissen des Lesers
  │  Beispiel  : „Wir haben’s ja“ (wenn Geld knapp ist)  ‖  „Das hast du ja toll hinbekommen!“
  │  Wirkung   : doppelbödig, verschlüsselt; in geschriebenen Texten nur durch Kontextanalyse erkennbar — kann belustigen, distanzieren oder verletzen
  └──────────────────────────────────────────────────────────

  ┌─ Oxymoron  [○ Oberstufe ]
  │  Erklärung : Verbindung zweier sich logisch widersprechender Begriffe in einer Wortgruppe.
  │  Beispiel  : „Hassliebe“  ‖  „beredtes Schweigen“  ‖  „alter Knabe“
  │  Wirkung   : widersprüchlich, spannungserzeugend
  └──────────────────────────────────────────────────────────

  ┌─ Paradoxon  [○ Oberstufe ]
  │  Erklärung : Scheinbar widersprüchliche Aussage, die bei näherem Hinsehen einen tieferen Sinn enthält.
  │  Beispiel  : „Ich weiß, dass ich nichts weiß“ (Sokrates)  ‖  „Weniger ist mehr“ (Mies van der Rohe)
  │  Wirkung   : scheinbar widersprüchlich, zum Nachdenken
  └──────────────────────────────────────────────────────────

  ┌─ Rhetorische Frage  [★ Mittelstufe]
  │  Erklärung : Eine Frage, auf die keine Antwort erwartet wird – die Antwort ist bereits klar.
  │  Beispiel  : „Wie viele Menschen müssen noch sterben?“  ‖  „Bin ich denn der Einzige, der das sieht?“
  │  Wirkung   : bindet Leser/Hörer ein — wer die Frage beantwortet, stimmt dem Sprecher bereits zu; erzeugt Zustimmungsdruck, Dringlichkeit oder Nachdenklichkeit
  └──────────────────────────────────────────────────────────

  ┌─ Sarkasmus  [★ Mittelstufe]
  │  Erklärung : Beißender, verletzender Spott – schärfer und gemeiner als Ironie.
  │  Beispiel  : „Gibt’s dein T-Shirt auch in deiner Größe?“  ‖  „Glanzleistung – wie immer!“
  │  Wirkung   : beißend spöttisch, verletzend
  └──────────────────────────────────────────────────────────

  ┌─ Katachrese  [○ Oberstufe ]
  │  Erklärung : Zwei Formen: (a) Tote Metapher — ein Bild, das durch Alltagsgebrauch nicht mehr als Bild wahrgenommen wird; (b) Bildbruch — zwei Metaphern werden widersprüchlich kombiniert
  │  Beispiel  : Tote Metapher: „Tischbein“  ‖  „Flaschenhals“  ‖  „Arm des Flusses“  ||  Bildbruch: „Das schlägt dem Fass die Krone ins Gesicht“
  │  Wirkung   : Tote Metapher: unauffällig, verblasst; Bildbruch: unfreiwillig komisch oder stilistisch ungeschickt — oft ein Fehler
  └──────────────────────────────────────────────────────────

╔══ SATZFIGUR (11 Stilmittel)

  ┌─ Ellipse  [★ Mittelstufe]
  │  Erklärung : Auslassung eines Satzteils, der gedanklich ergänzt werden kann.
  │  Beispiel  : „Je später der Abend, desto schöner die Gäste“  ‖  „Ende gut, alles gut.“
  │  Wirkung   : knapp, dramatisch, temporeich
  └──────────────────────────────────────────────────────────

  ┌─ Inversion  [○ Oberstufe ]
  │  Erklärung : Umstellung der üblichen Satzgliedfolge zur Hervorhebung.
  │  Beispiel  : „In der ungeheuren Weite / Reget keine Welle sich“ (Goethe)  ‖  „Dunkel war’s, der Mond schien helle“
  │  Wirkung   : hervorhebend durch Umstellung
  └──────────────────────────────────────────────────────────

  ┌─ Parallelismus  [★ Mittelstufe]
  │  Erklärung : Wiederholung der gleichen syntaktischen Struktur in aufeinanderfolgenden Sätzen oder Satzteilen.
  │  Beispiel  : „Heiß die Liebe, kalt der Tod“  ‖  „Ich kam, ich sah, ich siegte“ (Caesar)
  │  Wirkung   : ordnend, rhythmisch
  └──────────────────────────────────────────────────────────

  ┌─ Klimax  [★ Mittelstufe]
  │  Erklärung : Stufenweise Steigerung von Ausdruck zu Ausdruck.
  │  Beispiel  : „Er sei mein Freund, mein Engel, mein Gott“ (Schiller)  ‖  „Veni, vidi, vici“ (Caesar)
  │  Wirkung   : steigernd, eindringlich
  └──────────────────────────────────────────────────────────

  ┌─ Akkumulation  [★ Mittelstufe]
  │  Erklärung : Reihung von Begriffen, die einem gemeinsamen Oberbegriff zugeordnet werden können.
  │  Beispiel  : „Sonne, Mond und Sterne!“  ‖  „Nun ruhen alle Wälder, Vieh, Menschen, Städt und Felder“ (Gerhardt)
  │  Wirkung   : veranschaulicht, eindringlich
  └──────────────────────────────────────────────────────────

  ┌─ Asyndeton  [○ Oberstufe ]
  │  Erklärung : Reihung von Wörtern oder Satzteilen ohne verbindende Konjunktionen.
  │  Beispiel  : „Ein Rot, ein Grün, ein Grau vorbeigesendet“ (Rilke)  ‖  „Alles rennet, rettet, flüchtet“ (Schiller)
  │  Wirkung   : temporeich, aufzählend, drängend
  └──────────────────────────────────────────────────────────

  ┌─ Chiasmus  [★ Mittelstufe]
  │  Erklärung : Überkreuzstellung von syntaktisch oder inhaltlich einander entsprechenden Satzgliedern — die Reihenfolge wird in der zweiten Hälfte umgekehrt (Schema: A–B / B–A)
  │  Beispiel  : „Die Kunst ist lang – und kurz ist unser Leben“ (Goethe)
  │  Wirkung   : pointiert, einprägsam, symmetrisch — erzeugt den Eindruck von Ausgewogenheit oder Gegensätzlichkeit
  └──────────────────────────────────────────────────────────

  ┌─ Polysyndeton  [○ Oberstufe ]
  │  Erklärung : Reihung, bei der zwischen allen Gliedern die gleiche Konjunktion wiederholt wird.
  │  Beispiel  : „Einigkeit und Recht und Freiheit“  ‖  „Und es wallet und siedet und brauset und zischt“ (Schiller)
  │  Wirkung   : verlangsamend, aufzählend, feierlich
  └──────────────────────────────────────────────────────────

  ┌─ Parataxe  [★ Mittelstufe]
  │  Erklärung : Aneinanderreihung von Hauptsätzen (Satzreihe) ohne Unterordnung.
  │  Beispiel  : „Am Anfang schuf Gott Himmel und Erde. Und die Erde war wüst.“ (1. Mose)
  │  Wirkung   : klar, direkt, schlicht
  └──────────────────────────────────────────────────────────

  ┌─ Hypotaxe  [★ Mittelstufe]
  │  Erklärung : Verschachtelung von Haupt- und Nebensätzen (Satzgefüge) mit Unterordnung.
  │  Beispiel  : „An einem der letzten Maitage, das Wetter war sommerlich, bog ein Landauer ein …“ (Fontane)
  │  Wirkung   : komplex, ausschweifend, detailreich
  └──────────────────────────────────────────────────────────

  ┌─ Enjambement  [★ Mittelstufe]
  │  Erklärung : Zeilensprung – ein Satz wird über das Versende hinaus in der nächsten Zeile fortgesetzt. — Nur in Lyrik, nicht in Prosa
  │  Beispiel  : „Über allen Gipfeln / Ist Ruh“ (Goethe)  ‖  „Ich bin so wild nach deinem / Erdbeermund“ (Villon/Klammer)
  │  Wirkung   : fließend, drängend, spannungserzeugend — nur in Gedichten; in Prosatexten nicht anwendbar
  └──────────────────────────────────────────────────────────

╔══ KLANGFIGUR (6 Stilmittel)

  ┌─ Alliteration  [★ Mittelstufe]
  │  Erklärung : Wiederholung des gleichen Anfangslauts bei benachbarten Wörtern.
  │  Beispiel  : „Milch macht müde Männer munter“  ‖  „Mit Mann und Maus“  ‖  „Wer wirbt, wirbt Wähler“
  │  Wirkung   : betont, einprägsam, klingend
  └──────────────────────────────────────────────────────────

  ┌─ Anapher  [★ Mittelstufe]
  │  Erklärung : Wiederholung eines Wortes oder einer Wortgruppe am Anfang aufeinanderfolgender Sätze oder Verse.
  │  Beispiel  : „Kein Licht. Kein Ton.“  ‖  „Wer nie sein Brot mit Tränen aß, / Wer nie …“ (Goethe)
  │  Wirkung   : verstärkend, rhythmisch
  └──────────────────────────────────────────────────────────

  ┌─ Repetitio  [○ Oberstufe ]
  │  Erklärung : Wiederholung eines Wortes oder Ausdrucks innerhalb eines Satzes oder Verses.
  │  Beispiel  : „O Mutter! Was ist Seligkeit? O Mutter!“ (Bürger)  ‖  „Mein Vater, mein Vater“ (Goethe)
  │  Wirkung   : eindringlich, wiederholend
  └──────────────────────────────────────────────────────────

  ┌─ Epipher  [○ Oberstufe ]
  │  Erklärung : Wiederholung eines Wortes oder einer Wortgruppe am Ende aufeinanderfolgender Sätze oder Verse.
  │  Beispiel  : „Ich fordere Moral, du lebst Moral“  ‖  „Doch alle Lust will Ewigkeit, / will tiefe, tiefe Ewigkeit“ (Nietzsche)
  │  Wirkung   : verstärkend am Satzende
  └──────────────────────────────────────────────────────────

  ┌─ Tautologie  [○ Oberstufe ]
  │  Erklärung : Wiederholung desselben Sachverhalts mit anderen Worten oder bedeutungsgleichen Ausdrücken.
  │  Beispiel  : „voll und ganz“  ‖  „nie und nimmer“  ‖  „immer und ewig“
  │  Wirkung   : bekräftigend, redundant
  └──────────────────────────────────────────────────────────

  ┌─ Elision  [○ Oberstufe ]
  │  Erklärung : Auslassung eines Lautes oder einer Silbe, meist um das Versmaß einzuhalten. — Vor allem in Lyrik, erkennbar am Apostroph (’)
  │  Beispiel  : „in ew’gen Tau“ (statt: ewigen)  ‖  „Hab’ ich“ (statt: Habe ich)
  │  Wirkung   : Metrum erhaltend, klanglich weich — vor allem in älteren Gedichten; in moderner Prosa kaum vorhanden
  └──────────────────────────────────────────────────────────
